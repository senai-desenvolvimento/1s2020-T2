Assistente Importar Arquivo Simples para SQLImport Flat File to SQL Wizard

  • 6 minutos para ler

Para obter conteúdo relacionado ao Assistente de Importação e de Exportação, consulte Assistente de Importação e Exportação do SQL Server.For content related to the Import and Export Wizard, see SQL Server Import and Export Wizard.

O Assistente Importar Arquivo Simples fornece uma maneira simples de copiar dados de um arquivo simples (.csv, .txt) para uma nova tabela no seu banco de dados.Import Flat File Wizard is a simple way to copy data from a flat file (.csv, .txt) to a new table in your database. Esta visão geral descreve os motivos para usar esse assistente, como encontrá-lo e um exemplo simples as ser seguido.This overview describes the reasons for using this wizard, how to find this wizard, and a simple example to follow.

Por que usar esse assistente?Why would I use this wizard?

Esse assistente foi criado para melhorar a experiência de importação atual usando uma estrutura inteligente conhecida como PROSE (Program Synthesis using Examples).This wizard was created to improve the current import experience leveraging an intelligent framework known as Program Synthesis using Examples (PROSE). Para um usuário sem conhecimento especializado de domínio, a importação de dados normalmente pode ser uma tarefa complexa, propensa a erros e entediante.For a user without specialized domain knowledge, importing data can often be a complex, error prone, and tedious task. Esse assistente simplifica tanto o processo de importação de modo que é necessário apenas selecionar um arquivo de entrada e o nome exclusivo da tabela, deixando que a estrutura PROSE cuide do restante.This wizard streamlines the import process as simple as selecting an input file and unique table name, and the PROSE framework handles the rest.

O PROSE analisa os padrões de dados no arquivo de entrada para inferir os nomes das colunas, bem como os tipos, os delimitadores e muito mais.PROSE analyzes data patterns in your input file to infer column names, types, delimiters, and more. Essa estrutura aprende a estrutura do arquivo e faz todo o trabalho pesado, de modo que os usuários não precisem fazê-lo.This framework learns the structure of the file and does all of the hard work so users don't have to.

PrerequisitesPrerequisites

Esse recurso só está disponível no SSMS (SQL Server Management Studio) v17.3 ou posterior.This feature is only available on SQL Server Management Studio (SSMS) v17.3 or later. Verifique se você está usando a versão mais recente.Make sure you are using the latest version. É possível encontrar a versão mais recente aqui.You can find the latest version here.

IntroduçãoGetting Started

Para acessar o Assistente Importar Arquivo Simples, siga estas etapas:To access the Import Flat File Wizard, follow these steps:

  1. Abra o SQL Server Management Studio.Open SQL Server Management Studio.
  2. Conecte-se a uma instância do Mecanismo de Banco de Dados do Microsoft SQL Server ou do localhost.Connect to an instance of the SQL Server Database Engine or localhost.
  3. Expanda Bancos de Dados, clique com o botão direito do mouse em um banco de dados ("test" no exemplo abaixo), aponte para Tarefas e clique em Importar Arquivo Simples acima de Importar Dados.Expand Databases, right-click a database (test in the example below), point to Tasks, and click Import Flat File above Import Data.

Para saber mais sobre as diferentes funções do assistente, consulte o seguinte tutorial:To learn more about the different functions of the wizard, refer to the following tutorial:

TutorialTutorial

Para os fins deste tutorial, sinta-se à vontade para usar seu próprio arquivo simples.For the purposes of this tutorial, feel free to use your own flat file. Caso contrário, este tutorial usará o seguinte CSV do Excel, o qual você pode copiar.Otherwise, this tutorial is using the following CSV from Excel, which you are free to copy. Se você usar esse CSV, nomeie-o example.csv e salve-o como um CSV em um local fácil, como a área de trabalho.If you use this CSV, title it example.csv and make sure to save it as a csv in an easy location such as your desktop.

Etapa 1: Acessar o assistente e a página IntroduçãoStep 1: Access Wizard and Intro Page

Acesse o assistente conforme descrito aqui.Access the wizard as described here.

A primeira página do assistente é a página inicial.The first page of the wizard is the welcome page. Se não quiser ver esta página novamente, clique em Não mostrar esta página inicial novamente.If you do not want to see this page again, feel free to click Do not show this starting page again.

Etapa 2: Especificar Arquivo de EntradaStep 2: Specify Input File

Clique em Procurar para selecionar o arquivo de entrada.Click browse to select your input file. Como padrão, o assistente procura arquivos .csv e .txt.At default, the wizard searches for .csv and .txt files.

O novo nome da tabela deverá ser exclusivo, caso contrário, o assistente não permitirá a continuação do processo.The new table name should be unique, and the wizard does not allow you to move further if not.

Etapa 3: Visualizar DadosStep 3: Preview Data

O assistente gera uma visualização na qual é possível exibir as primeiras 50 linhas.The wizard generates a preview that you can view for the first 50 rows. Se houver problemas, clique em Cancelar, caso contrário, prossiga para a próxima página.If there are any problems, click cancel, otherwise proceed to the next page.

Etapa 4: Modificar ColunasStep 4: Modify Columns

O assistente identifica o que ele acredita que são os nomes corretos das colunas, bem como os tipos de dados, etc. Aqui será possível editar os campos se eles estiverem incorretos (por exemplo, o tipo de dados deve ser um float em vez de um int).The wizard identifies what it believes are the correct column names, data types, etc. Here is where you can edit the fields if they are incorrect (for example, data type should be a float instead of an int).

Continue quando estiver pronto.Proceed when ready.

Etapa 5: ResumoStep 5: Summary

Esta é uma página de resumo que exibe a configuração atual.This is simply a summary page displaying your current configuration. Se houver problemas, será possível voltar às seções anteriores.If there are issues, you can go back to previous sections. Caso contrário, clicar em Concluir iniciará a tentativa de execução do processo de importação.Otherwise, clicking finish attempts the import process.

Etapa 6: ResultadosStep 6: Results

Esta página indica se a importação foi bem-sucedida.This page indicates whether the import was successful. Se uma marca de seleção verde aparecer, a operação foi bem-sucedida, caso contrário, talvez seja necessário examinar a configuração ou o arquivo de entrada em busca de erros.If a green check mark appears, it was a success, otherwise you may need to review your configuration or input file for any errors.
