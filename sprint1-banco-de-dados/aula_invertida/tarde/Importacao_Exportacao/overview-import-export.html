Importar e exportar dados do SQL Server e do Banco de Dados SQL do AzureImport and export data from SQL Server and Azure SQL Database

  • 10/27/2017
  • 5 minutos para ler

Neste artigo

APLICA-SE A: SQL Server Banco de Dados SQL do Azure Azure Synapse Analytics (SQL DW) Parallel Data Warehouse APPLIES TO: SQL Server Azure SQL Database Azure Synapse Analytics (SQL DW) Parallel Data Warehouse

Use uma variedade de métodos para importar e exportar dados do SQL Server e do Banco de Dados SQL do Azure.You can use a variety of methods to import data to, and export data from, SQL Server and Azure SQL Database. Esses métodos incluem instruções Transact-SQL, ferramentas de linha de comando e assistentes.These methods include Transact-SQL statements, command-line tools, and wizards.

Também importe e exporte dados em uma variedade de formatos.You can also import and export data in a variety of data formats. Esses formatos incluem arquivos simples, Excel, os principais bancos de dados relacionais e vários serviços de nuvem.These formats include flat files, Excel, major relational databases, and various cloud services.

Métodos para importar e exportar dadosMethods for importing and exporting data

Usar instruções Transact-SQLUse Transact-SQL statements

Importe dados com os comandos BULK INSERT ou OPENROWSET(BULK...).You can import data with the BULK INSERT or the OPENROWSET(BULK...) commands. Normalmente, você executa esses comandos no SSMS (SQL Server Management Studio).Typically you run these commands in SQL Server Management Studio (SSMS). Para obter mais informações, consulte Importar dados em massa usando BULK INSERT ou OPENROWSET(BULK...).For more info, see Import Bulk Data by Using BULK INSERT or OPENROWSET(BULK...).

Usar o BCP no prompt de comandoUse BCP from the command prompt

Importe e exporte dados com o utilitário de linha de comando BCP.You can import and export data with the BCP command-line utility. Para obter mais informações, consulte Importar e exportar dados em massa usando o utilitário BCP.For more info, see Import and Export Bulk Data by Using the bcp Utility.

Usar o Assistente Importar Arquivo SimplesUse the Import Flat File Wizard

Caso não precise de todas as opções de configuração disponíveis no Assistente de Importação e Exportação e em outras ferramentas, você pode importar um arquivo de texto para o SQL Server usando o Assistente Importar Arquivo Simples no SQL Server Management Studio.If you don't need all the configuration options available in the Import and Export Wizard and other tools, you can import a text file into SQL Server by using the Import Flat File Wizard in SQL Server Management Studio (SSMS). Para obter mais informações, confira os seguintes artigos:For more info, see the following articles:

  • Assistente Importar Arquivo Simples para SQLImport Flat File to SQL Wizard
  • Novidades do SQL Server Management Studio 17.3 What's new in SQL Server Management Studio 17.3
  • Apresentamos os novo Assistente Importar Arquivo Simples no SQL Server Management Studio 17.3Introducing the new Import Flat File Wizard in SSMS 17.3

Usar o Assistente para Importação e Exportação do SQL ServerUse the SQL Server Import and Export Wizard

Importe ou exporte dados de uma variedade de fontes e destinos com o Assistente para Importação e Exportação do SQL Server.You can import data to, or export data from, a variety of sources and destinations with the SQL Server Import and Export Wizard. Para usar o assistente, você deve ter o SSIS (SQL Server Integration Services) ou o SSDT (SQL Server Data Tools) instalado.To use the wizard, you must have SQL Server Integration Services (SSIS) or SQL Server Data Tools (SSDT) installed. Para obter mais informações, consulte Importar e exportar dados com o Assistente para Importação e Exportação do SQL Server.For more info, see Import and Export Data with the SQL Server Import and Export Wizard.

Criar sua própria importação ou exportaçãoDesign your own import or export

Se desejar criar uma importação de dados personalizada, use um dos seguintes recursos ou serviços:If you want to design a custom data import, you can use one of the following features or services:

  • SQL Server Integration Services.SQL Server Integration Services. Para obter mais informações, consulte SQL Server Integration Services.For more info, see SQL Server Integration Services.
  • Azure Data Factory.Azure Data Factory. Para obter mais informações, consulte Introdução ao Azure Data Factory.For more info, see Introduction to Azure Data Factory.

Formatos de dados para importação e exportaçãoData formats for import and export

Formatos com suporteSupported formats

Importe e exporte dados para arquivos simples ou uma variedade de outros formatos de arquivo, bancos de dados relacionais e serviços de nuvem.You can import data from, and export data to, flat files or a variety of other file formats, relational databases, and cloud services. Para saber mais sobre essas opções de ferramentas específicas, consulte os seguintes tópicosTo learn more about these options for specific tools, see the following topics

  • Para o Assistente para Importação e Exportação do SQL Server, consulte Conectar-se a fontes de dados com o Assistente para Importação e Exportação do SQL Server.For the SQL Server Import and Export Wizard, see Connect to Data Sources with the SQL Server Import and Export Wizard.
  • Para o SQL Server Integration Services, consulte Conexões do SSIS (Integration Services).For SQL Server Integration Services, see Integration Services (SSIS) Connections.
  • Para o Azure Data Factory, consulte Conectores do Azure Data Factory.For Azure Data Factory, see Azure Data Factory Connectors.

Formatos de dados usados com frequênciaCommonly used data formats

Há considerações especiais e exemplos disponíveis para alguns formatos de dados usados com frequência.There are special considerations and examples available for some commonly-used data formats. Para saber mais sobre esses formatos de dados, consulte os seguintes tópicos:To learn more about these data formats, see the following topics:

  • Para o Excel, consulte Importar do Excel.For Excel, see Import from Excel.
  • Para o JSON, consulte Importar documentos JSON.For JSON, see Import JSON Documents.
  • Para o XML, consulte Importar e exportar documentos XML.For XML, see Import and Export XML Documents.
  • Para o Armazenamento de Blobs do Azure, consulte Importar e exportar do Armazenamento de Blobs do Azure.For Azure Blob Storage, see Import and Export from Azure Blob Storage.

Próximas etapasNext steps

Caso não tenha certeza de onde começar com a tarefa de importação ou exportação, considere a possibilidade de usar o Assistente para Importação e Exportação do SQL Server.If you're not sure where to begin with your import or export task, consider the SQL Server Import and Export Wizard. Para obter uma introdução rápida, consulte Começar com este exemplo simples do Assistente para Importação e Exportação.For a quick introduction, see Get started with this simple example of the Import and Export Wizard.
